# ساخت فایل نصبی ویندوز (.exe) با Electron + Vite

این پروژه یک اپلیکیشن React (Vite) است. با اضافه کردن Electron و electron-builder
می‌توانید خروجی نصبی ویندوز (.exe) بسازید.

## پیش‌نیازها
- ویندوز ۱۰ یا ۱۱
- نصب Node.js LTS (از سایت nodejs.org)
- فضای کافی دیسک

## حالت توسعه (Dev)
```bash
npm install
npm run electron:dev
```
با اجرای دستور بالا، Vite Dev Server بالا می‌آید و Electron پنجره دسکتاپ را باز می‌کند.

## ساخت خروجی تولیدی (Production) و فایل نصبی
1) ابتدا خروجی Vite را بسازید:
```bash
npm run build
```
2) سپس بسته نصبی ویندوز را بسازید:
```bash
npm run electron:build
```
خروجی در پوشه‌ی `dist/` (فایل‌های وب) و فایل نصبی در پوشه‌ی `dist/` تولیدشده توسط electron-builder (معمولاً `dist/*.exe` یا زیرپوشه `dist/` و `release/`) قرار می‌گیرد.

> نکته: اگر آیکون سفارشی دارید (ico.)، پوشه‌ی `build/` بسازید و فایل آیکون را داخلش قرار دهید و تنظیمات `build` را به‌روزرسانی کنید.

## نکات
- اگر فایروال/آنتی‌ویروس هنگام اجرای فایل نصبی هشدار داد، ناشی از نبود امضای دیجیتال است. برای انتشار عمومی، بهتر است کد را Sign کنید.
- در صورت نیاز به قابلیت‌های سیستمی بیشتر، می‌توانیم از `preload.js` و `contextBridge` استفاده کنیم.
